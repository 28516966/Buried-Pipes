# Buried-Pipes
Analysis tools for the design of buried pipes. 

## Traffic Pressures
Executable file:  'buried_pipes_traffic_pressures.exe'

Enables the calculation of live loading traffic surcharge Ps for structural design of buried pipes where pressures may be calculated for any arbitrary set of point loads. 
Utility is also provided to enable fast discretisation of wheel contact pressures into a set of point loads.
